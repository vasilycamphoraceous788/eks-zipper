## 📦 What is eks-zipper?

eks-zipper is a Windows tool that helps you compress and extract files with common archive formats. It works with ZIP, 7Z, RAR, TAR, GZ, BZ2, and XZ files. If you often deal with downloaded archives, backups, or shared folders, this tool gives you a simple way to open and pack files without using several apps.

It is made for everyday use. You can use it to open an archive, unpack files, or create a compressed file for storage and sharing.

## 🧪 Common file behavior

Some archive files look similar but act in different ways:

- `.zip` files usually open fast and are common on Windows
- `.rar` files may contain many files in one package
- `.tar` files often hold folders without compression by themselves
- `.gz`, `.bz2`, and `.xz` files are often used for compressed data
- `.7z` files usually give strong compression

eks-zipper is made to handle these cases in one place.
